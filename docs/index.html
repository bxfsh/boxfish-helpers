after

Returns all of the items in the collection after the specified count.

Params:

  • array - (Array) - Collection
  • count - (Number) - Number of items to exclude
{{after posts 5}}

and

And operator helper

Params:

  • a - (String)
  • b - (String) 
{{#and isAdmin isLoggedIn}}
// Do Something
{{/and}}

arrayContains

Determines if an array contains an element with specified value

Params:

  • array - (Array)
  • value - (String) 
{{#arrayContains countries 'Ireland'}}
// Do Something
{{/arrayContains}}

before

Returns all of the items in the collection before the specified count.

Params:

  • array - (Array)
  • count - (Number) 
{{before posts 10}}

capitalize

Capitalize a string

Params:

  • string - (String) 
{{capitalize firstName}}

commaSeparate

Returns an array list separated by commas

Params:

  • array - (Array) 
{{commaSeparate genres}}

compare

Compare two values

Params:

  • left - (Number)
  • operator - (String)
  • right - (Number) 
{{#compare 1 '>=' 5}}
// Do Something
{{/compare}}

dateHasPast

Determines if a date is in the past

Params:

  • date - (Date) 
{{#dateHasPast date}}
// Do Something
{{/dateHasPast}}

defaultValue

Returns default value if value is missing

Params:

  • val - (String)
  • defaultVal - (String) 
{{defaultValue name 'unknown'}}

eachProperty

Returns an array of items with specified property

Params:

  • context - (Object)

eq

Determines if one value equals another

Params:

  • val - (String|Number)
  • match - (String|Number) 
{{#eq 'one' 'two'}}
// Do Something
{{/eq}}

exists

Determine if value exists

Params:

  • val - (String) 
{{#exists someValue}}
// Do Something
{{/exists}}

first

Returns a sliced array from 0 to a specified position

Params:

  • array - (Array)
  • count - (Number) 
{{first posts 10}}

formatDate

Format a date string with moment

Params:

  • date - (String)
  • format - (String) (optional)
{{formatDate date 'hh:mm'}}

formatTweet

Format a tweet to include links for urls, hashtags and users

Params:

  • tweet - (String) 
{{{formatTweet tweetText}}} // Note: triple braces

formatTwitterSearch

Finds hashtags and handles in a string and adds triggers for twitter search

Params:

  • tweet - (String) 
{{{formatTwitterSearch tweet}}} // Note the triple braces

fromNow

Returns the time difference between now and a specified date

Params:

  • date - (Date) 
{{fromNow date}}

gt

Greater than operator

Params:

  • a - (String|Number)
  • b - (String|Number) 
{{gt 4 5}}

gte

Greater than or equal to operator

Params:

  • a - (String|Number)
  • b - (String|Number) 
{{gt 4 5}}

gtlt

Greater than or less than operator

Params:

  • a - (String|Number)
  • b - (String|Number) 
{{gt 4 5}}

hasProperty

Returns array of items with specific property

Params:

  • array - (Array)
  • prop - (String)
  • val - (String) 
{{#hasProperty movies 'genre' 'horror'}}
// Horror Movies
{{/hasProperty}}

lowercase

Return string in lowercase

Params:

  • str - (String) 
{{lowercase 'TEXT'}} // Outputs 'text'

or

OR operator

Params:

  • a - (String|Number)
  • b - (String|Number) 
{{#or true false}}
// Do Something
{{/or}}

pluralize

Returns a pluralized version of a string

Params:

  • array - (Array)
  • string - (String) (expects singular version)
{{pluralize 5 'post'}} // Outputs '5 posts'

round

Math Round helper

Params:

  • val - (Number) 
{{round 1.3}} // Outputs 1

slugify

Returns a slugified version of a string

Params:

  • str - (String) 
{{slugify 'The Walking Dead'}} // Outputs 'the-walking-dead'

stringify

Stringify JSON

Params:

  • json - (Object) 
{{{stringify json}}} // Note the triple braces

today

Returns todays date

Params:

  • format - (String) 
{{today 'Do MMM, YYYY'}}

truncate

Truncate a string to specific length

Params:

  • paragraph - (String)
  • wordwise - (Boolean)
  • max - (Number)
  • tail - (String) 
{{truncate paragraph true 100 '...'}}

uppercase

Returns string in uppercase

Params:

  • str - (String) 
{{uppercase 'boxfish'}} // Outputs 'BOXFISH'
